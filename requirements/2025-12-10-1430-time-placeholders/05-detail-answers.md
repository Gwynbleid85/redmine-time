# Detail Answers

## Q1: Should users be able to add a placeholder directly from the "Add Task" button, or should there be a separate "Add Placeholder" button?
**Answer:** Separate button
**Notes:** Keeps the workflows distinct and avoids confusion

## Q2: Should placeholders support editing after creation (change date, duration, type)?
**Answer:** Yes
**Notes:** Users may make mistakes or need to adjust

## Q3: Should the placeholder dialog allow selecting a date range for multi-day entries (e.g., 5 vacation days), or require adding each day individually?
**Answer:** Individual days
**Notes:** Simpler implementation, consistent with time entries

## Q4: When a day has less than 8 hours (shown in red), should placeholder hours count toward making it "green" (8+ hours)?
**Answer:** Yes
**Notes:** The goal is to show total monthly time including non-work

## Q5: Should there be a way to quickly add common placeholders (e.g., "Add full day vacation" button)?
**Answer:** No
**Notes:** Keep UI simple, defaults already handle common cases
